# anyHouse

🔥 仿 ClubHouse 核心语音模块 。

👉 基于 anyRTC RTC SDK 实现高音质语音体验。

👉 基于 anyRTC RTM SDK 实现房间内举手/邀请/人员列表等。

🥰 代码简洁，逻辑清晰。

👀 使用 ViewModel,LiveData，数据与页面分离。

🌲 使用 DiffUtil 优化列表。

🌝 使用 Kotlin 协程优化异步任务。

⏬ [下载体验](https://www.pgyer.com/dccE) ⏬
